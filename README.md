# Parcial Arep Tercio 1
Construir un servicio web que permita almacenar llaves con su valor y consultarlas.

## Instalacion

```
mvn clean install
```

## Ejecuci√≥n

Clase del HttpServer
```

```

```
java -cp .\target\classes org.eci.arep.t1.FacadeServer
```
